# End-to-End-5G-Network-Planning-AI-Optimization-MIMO-Antenna-for-Mobile-Communications-
This project offers an end-to-end 5G solution for Dokki, combining HFSS-designed 8-port MIMO antennas with Atoll-based network planning. It features an AI engine (Q-Learning/DNN) for optimization and a Flutter-based "TowerGuard" app for real-time monitoring and smart tower management
