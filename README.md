5G End-to-End Network Planning & AI OptimizationOverviewThis project presents an integrated solution for 5G network deployment in the Dokki area, bridging the gap between electromagnetic antenna design and intelligent network planning. 
The system utilizes a custom-designed 8-port MIMO antenna system and an AI-driven engine for optimized performance.
Key Features:
MIMO Antenna Design: High-performance 8-port antenna system designed and simulated in Ansys HFSS. 
Network Planning: Urban network planning for the Dokki area using Atoll to analyze RSRP, SINR, and throughput.  AI Optimization: Intelligent engine utilizing Q-Learning and Deep Neural Networks (DNN) for antenna placement and forecasting.
TowerGuard App: A Flutter-based mobile application with Firebase integration for real-time monitoring and smart tower management.
Tech StackSimulation: Ansys HFSS, Atoll. 
Programming: Python (Q-Learning, DNN, Scikit-learn), FastAPI/Streamlit.
